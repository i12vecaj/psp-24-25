Para crear el proyecto en IntelliJ, primero generé un proyecto Maven en Java e importé la biblioteca Commons. 
Luego, añadí las clases al nuevo proyecto.

Para que el sistema funcione correctamente, es necesario reemplazar todos los usuarios y contraseñas en las clases por tu usuario y contraseña de Windows.
Después, ejecuta el servidor, que te solicitará seleccionar un directorio para subir los archivos.

Para realizar las distintas operaciones, ejecuta la clase ClienteFicheros, desde donde podrás gestionar todas las funciones disponibles.
