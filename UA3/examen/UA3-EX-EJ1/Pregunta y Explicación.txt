Pregunta para el estudiante:
¿Es más conveniente utilizar TCP o UDP en este caso? Explica tu respuesta.
TCP es mejor porque, garantiza la entrega, mantiene el orden de los mensajes y permite conexiones estables con múltiples clientes. 
UDP es más rápido, pero puede perder mensajes o enviarlos desordenados, lo que no es ideal para un chat.

Expliación del Código:
El código tiene una estructura básica de un servidor y un cliente.
El servidor lo levantamos con una terminal (java ServidorChat) cuando este este activo, abrimos otra terminal a parte,
para ingresar los usuarios (java ClienteChat) y poder comunicarse con el servidor.
Para ver el funcinamiento completo, recomiendo abrir 3 terminales, una para el servidor y dos para los clientes, 
y verás como en la terminal del servidor se muestran los mensajes de los clientes.
