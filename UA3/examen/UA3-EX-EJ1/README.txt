¿Es más conveniente utilizar TCP o UDP en este caso? Explica tu respuesta.


Realmente en este caso es una cuestion de preferencias y de uso

La principal diferencia entre el TCP  y el UDP  es que el TCP es un protocolo basado en conexiones y el UDP es sin conexiones TCP es más fiable, transfiere los datos mas lento. El UDP es menos fiable pero funciona más rápido. Esto hace que cada protocolo sea adecuado para distintos tipos de transferencia de datos.

El TCP y el UDP son dos métodos diferentes para hacer el mismo trabajo: transferir datos a través de Internet. Permiten que los servidores y dispositivos se comuniquen para que pueda enviar correos un mensaje por Telegram, ver Youtube, jugar , etc....

A si que creo que es mas conveniente usar UDP porque : en este tipo de conexiones queremos la mayor velocidad posible
