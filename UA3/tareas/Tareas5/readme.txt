Compila las clases Numeros, Servidor y Cliente .
Primero, ejecuta la clase Servidor en una terminal para iniciar el servidor.
Luego, ejecuta la clase Cliente en otra terminal para interactuar con el servidor.
Introduce un numero entero cuando se te solicite en el cliente. Si el numero es negativo, recibiras un mensaje de error.
Si el numero es valido, el servidor calculara el cuadrado y el cubo, y el cliente mostrara los resultados.
