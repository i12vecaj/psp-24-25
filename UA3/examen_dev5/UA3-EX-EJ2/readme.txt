Ejercicio 2: Sistema de Notificaciones Rápidas
Una empresa quiere implementar un sistema de alertas rápidas para sus empleados. El servidor debe enviar mensajes cortos de notificación a múltiples clientes sin necesidad de una conexión persistente. No es crítico que todas las notificaciones lleguen en orden o sean confirmadas, pero sí que sean enviadas de manera eficiente.
 Requisitos:
El servidor enviará notificaciones a los clientes conectados cuando reciba una nueva alerta desde una fuente externa.
Los clientes podrán escuchar mensajes sin necesidad de establecer una conexión persistente.
No se espera que los clientes respondan al servidor.
El servidor puede enviar la misma notificación a múltiples clientes simultáneamente.
 Pregunta para el estudiante:
¿Es más conveniente utilizar TCP o UDP en este caso? Explica tu respuesta.
